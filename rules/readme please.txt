Использование
Смотрите прилагаемое руководство страницу вверх-к-дата, полных инструкции по использованию.
Использование:
./stm32flash [-bvngfhc] [- [RW] имя_файла] / DEV / ttyS0
-b Скорость передачи Скорость (по умолчанию 57600)
-r имя файла чтения флэш-файл, чтобы
-w имя файла записи флэш подать
-u Отключить флэш защиту от записи
-en только стереть п страницы, прежде чем писать вспышки
-v Проверьте пишет
-n число попыток не удалось, пишет до подсчет раз (по умолчанию 10)
-g адрес выполнение Начало указанному адресу (0 = флэш начало)
-s start_page флэш на указанная страница (0 = флэш начало)
-f Принудительно двоичный парсер
-h Показать эту помощь
-с резюме соединение (не отправлять первоначальный INIT)
        * Скорость передачи должна быть такой же, как в первый инициализации *
        Это полезно, если сброс не
Примеры:
Получить информацию об устройстве:
    ./stm32flash / DEV / ttyS0

Пишите проверить, а затем начать выполнение:
    ./stm32flash -w -v файла -g 0x0 / DEV / ttyS0

Читайте вспышку в файл:
    ./stm32flash -r имя файла / DEV / ttyS0

Начните выполнение:
    ./stm32flash -g 0x0 / DEV / ttyS0
